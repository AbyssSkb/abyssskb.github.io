库仑定律

定义 q1q_1q1​ 对 q2q_2q2​ 施加的力

F⃗=14πε0q1q2r2er⃗\vec{F} = \frac{1}{4\pi\varepsilon_0} \frac{q_1 q_2}{r^2} \vec{e_r} F=4πε0​1​r2q1​q2​​er​​

其中 er⃗\vec{e_r}er​​ 是由 q1q_1q1​ 指向 q2q_2q2​ 的单位向量

电场强度

描述电场中各点电场强弱和方向的物理量

E⃗=F⃗q0\vec{E} = \frac{\vec{F}}{q_0} E=q0​F​

具体到静电场中，拿点电荷来举例

E⃗=F⃗q0=14πε0qr2er⃗\vec{E} = \frac{\vec{F}}{q_0} = \frac{1}{4\pi\varepsilon_0}\frac{q}{r^2} \vec{e_r} E=q0​F​=4πε0​1​r2q​er​​

根据力的可叠加性，由此可以计算复杂带电体在空间某一点的电场强度

下面列出一些常见带电体的公式

电势能

由功能原理，静电场力对电荷所做的功就等于电荷电势能增量的负值

WAB=−(EpB−EpA)=EpA−EpBW_{AB} = -(E_{pB} - E_{pA}) = E_{pA} - E_{pB} WAB​=−(EpB​−EpA​)=EpA​−EpB​

又因为

WAB=∫ABq0E⃗⋅ ⁣dl⃗W_{AB} = \int_{AB} q_0 \vec{E}\cdot \mathop{}\!\mathrm{d}\vec{l} WAB​=∫AB​q0​E⋅dl

令 EpB=0E_{pB} = 0EpB​=0，则

EpA=∫ABq0E⃗⋅ ⁣dl⃗E_{pA} = \int_{AB} q_0 \vec{E} \cdot \mathop{}\!\mathrm{d}\vec{l} EpA​=∫AB​q0​E⋅dl

物理意义：把试验电荷 q0q_0q0​ 从 AAA 点移动到零电势能点处静电场力所做的功

静电场的环路定理

由于静电力是保守力，因此静电力做功仅于 q0q_0q0​ 的始末位置有关，绕一圈的做功为零，由此可推出静电场的环路定理。

∮lE⃗⋅ ⁣dl⃗=0\oint_l \vec{E} \cdot\mathop{}\!\mathrm{d}\vec{l} = 0 ∮l​E⋅dl=0

电势

令 VB=0V_B = 0VB​=0，则 AAA 点电势

VA=∫ABE⃗⋅ ⁣dl⃗V_A = \int_{AB} \vec{E} \cdot\mathop{}\!\mathrm{d}\vec{l} VA​=∫AB​E⋅dl

UAB=VA−VB=∫ABE⃗⋅ ⁣dl⃗U_{AB} = V_A - V_B = \int_{AB} \vec{E} \cdot\mathop{}\!\mathrm{d}\vec{l} UAB​=VA​−VB​=∫AB​E⋅dl

拿点电荷的电场来举例，令 V∞=0V_\infty = 0V∞​=0，则

V=∫r∞q4πε0r2er⃗⋅ ⁣dl⃗=∫r∞q ⁣dr4πε0r2=q4πε0r\begin{split} V &= \int_r^\infty \frac{q}{4\pi\varepsilon_0r^2}\vec{e_r}\cdot\mathop{}\!\mathrm{d}\vec{l} \\ &= \int_r^\infty \frac{q\mathop{}\!\mathrm{d}r}{4\pi\varepsilon_0r^2} \\ &= \frac{q}{4\pi\varepsilon_0r} \end{split} V​=∫r∞​4πε0​r2q​er​​⋅dl=∫r∞​4πε0​r2qdr​=4πε0​rq​​

p⃗=qr0⃗\vec{p} = q \vec{r_0} p​=qr0​​

r0⃗\vec{r_0}r0​​ 为由 −q-q−q 指向 qqq 的距离向量

若 x≫r0x \gg r_0x≫r0​，则

E⃗=14πε02p⃗x3\vec{E} = \frac{1}{4\pi\varepsilon_0}\frac{2\vec{p}}{x^3} E=4πε0​1​x32p​​

在匀强电场中受到的力

F⃗=0⃗\vec{F} = \vec{0} F=0

Ep=−p⃗⋅E⃗E_p = - \vec{p} \cdot \vec{E} Ep​=−p​⋅E

由能量最低原理也可以推断出电偶极子总是趋向于向电场的方向旋转

静电平衡

条件

  1. 导体内任一点的电场强度都等于零
  2. 导体表面处电场强度的方向，都与导体表面垂直

推论

  1. 导体内任意两点间的电势是相等的
  2. 导体内部处处没有净电荷存在，净电荷只能分布于导体的表面上
  3. 导体表面附近场强的大小与该处电荷的面密度成正比
  4. 腔内无电荷时，导体的净电荷只能分布在外表面

应用

屏蔽外电场

空腔导体外的电场不会影响空腔内部的电场分布

屏蔽内电场

接地空腔导体内的电场不会影响空腔外的电场

静电场中的电介质

极化电荷的场

E⃗=E0⃗+E′⃗\vec{E} = \vec{E_0} + \vec{E'} E=E0​​+E′

实验表明

E0⃗=εrE⃗\vec{E_0} = \varepsilon_r \vec{E} E0​​=εr​E

电极化强度

介质中某一点的电极化强度矢量等于该点附近单位体积中的分子电距的矢量和

P⃗=∑p⃗ΔV\vec{P} = \frac{\sum\vec{p}}{\Delta V} P=ΔV∑p​​

均匀介质极化时只在介质表面出现极化电荷，因此可以得到

P=σ′ΔSlΔSl=σ′P = \frac{\sigma' \Delta Sl}{\Delta Sl} = \sigma' P=ΔSlσ′ΔSl​=σ′

而

E′=σ′ε0E' = \frac{\sigma'}{\varepsilon_0} E′=ε0​σ′​

所以

E′⃗=−P⃗ε0\vec{E'} = -\frac{\vec{P}}{\varepsilon_0} E′=−ε0​P​

有介质时的高斯定理

由高斯定理

∯SE⃗⋅ ⁣dS⃗=1ε0(Q0+Q′)\oiint\limits_S \vec{E} \cdot\mathop{}\!\mathrm{d}\vec{S} = \frac{1}{\varepsilon_0}(Q_0 + Q') S∬​​E⋅dS=ε0​1​(Q0​+Q′)

而

Q′=−∯SP⃗⋅ ⁣dS⃗Q' = -\oiint\limits_S \vec{P} \cdot\mathop{}\!\mathrm{d}\vec{S} Q′=−S∬​​P⋅dS

所以

∯S(ε0E⃗+P⃗)⋅ ⁣dS⃗=Q0\oiint\limits_S \left(\varepsilon_0\vec{E} + \vec{P}\right) \cdot\mathop{}\!\mathrm{d} \vec{S} = Q_0 S∬​​(ε0​E+P)⋅dS=Q0​

我们定义电位移矢量

D⃗=ε0E⃗+P⃗\vec{D} = \varepsilon_0\vec{E} + \vec{P} D=ε0​E+P

和电位移通量

∯SD⃗⋅ ⁣dS⃗\oiint\limits_S \vec{D} \cdot\mathop{}\!\mathrm{d}\vec{S} S∬​​D⋅dS

则

∯SD⃗⋅ ⁣dS⃗=Q0\oiint\limits_S \vec{D} \cdot\mathop{}\!\mathrm{d}\vec{S} = Q_0 S∬​​D⋅dS=Q0​

我们称之为有介质时的高斯定理，通过任意闭合曲面的电位移通量等于这个闭合曲面所包围的自由电荷的代数和

再来看一下电位移矢量和电场强度的关系

D⃗=ε0E⃗+P⃗=ε0E⃗−ε0E′⃗=ε0E0⃗=ε0εrE⃗=εE⃗\vec{D} = \varepsilon_0 \vec{E} + \vec{P} = \varepsilon_0 \vec{E} -\varepsilon_0 \vec{E'} = \varepsilon_0 \vec{E_0} = \varepsilon_0 \varepsilon_r \vec{E} = \varepsilon \vec{E} D=ε0​E+P=ε0​E−ε0​E′=ε0​E0​​=ε0​εr​E=εE

故

E⃗=D⃗ε\vec{E} = \frac{\vec{D}}{\varepsilon} E=εD​

所以有介质时的高斯定理又可以改写成

∯SE⃗⋅ ⁣dS⃗=Q0ε\oiint\limits_S \vec{E}\cdot\mathop{}\!\mathrm{d}\vec{S} = \frac{Q_0}{\varepsilon} S∬​​E⋅dS=εQ0​​

只需要把高斯定理中的 ε0\varepsilon_0ε0​ 改成 ε\varepsilonε 即可
